Use this simple install command to install Warehouse NAV on your Raspberry Pi!
```bash
curl -fsSL https://raw.githubusercontent.com/RX-LOST/Warehouse-NAV/main/install.sh | sudo bash
```
Note: This version only works on 64-bit Linux, such as amd64 and arm64 architectures.

For Devs:
All changes made to the repository trigger an automatic build and release for Pi downloads of the built tool.
